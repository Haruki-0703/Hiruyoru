# ヒルヨル - 写真1枚で終わる食事報告

**Version 0.5.0** - 買い物リスト生成 & 週間レポート機能追加

[![Changelog](https://img.shields.io/badge/CHANGELOG-View-blue)](CHANGELOG.md)

毎日のお昼ご飯を記録し、夜ご飯の献立を提案してくれるモバイルアプリです。

## 概要

「ヒルヨル」は、忙しい毎日の食事管理をシンプルにするアプリです。ランチの写真を撮るだけでAIが自動で料理を認識し、その情報を元に栄養バランスを考慮した夜ご飯のメニューを提案します。

## 主な機能

### 基本機能（ゲストモード対応）
- **ランチ記録**: 写真撮影またはテキスト入力で簡単に記録
- **履歴閲覧**: 過去の食事記録を確認

### プレミアム機能（ログイン必須）
- **AI画像解析**: 写真から料理名・カテゴリを自動認識
- **おすすめディナー**: ランチの内容に基づいたAIによる夜ご飯提案
- **買い物リスト生成**: おすすめディナーから必要な食材を自動生成
- **週間レポート**: 1週間の食事傾向をグラフで可視化し、栄養バランスのアドバイスを提供
- **クラウド同期**: 複数デバイス間でのデータ同期

## 技術スタック

- **フロントエンド**: React Native (Expo SDK 54)
- **バックエンド**: Node.js + tRPC
- **データベース**: MySQL (TiDB)
- **AI**: LLM統合（画像解析・レコメンデーション）
- **認証**: Manus OAuth

## プロジェクト構造

```
├── app/                    # Expo Router画面
│   ├── (tabs)/            # タブナビゲーション
│   │   ├── index.tsx      # ホーム画面
│   │   ├── record.tsx     # ランチ記録画面
│   │   ├── recommend.tsx  # おすすめディナー画面
│   │   ├── reports.tsx    # 週間レポート画面
│   │   └── history.tsx    # 履歴画面
│   └── modal.tsx          # ログイン画面
├── components/            # 共通コンポーネント
├── constants/             # テーマ・定数
├── hooks/                 # カスタムフック
├── server/                # バックエンドAPI
│   ├── routers.ts         # tRPCルーター
│   ├── db.ts              # データベースクエリ
│   └── storage.ts         # S3ストレージ
├── drizzle/               # データベーススキーマ
└── assets/                # 画像・アイコン
```

## セットアップ

### 必要条件
- Node.js 22.x
- pnpm

### インストール

```bash
# 依存関係のインストール
pnpm install

# 開発サーバーの起動
pnpm dev
```

### データベースマイグレーション

```bash
pnpm db:push
```
