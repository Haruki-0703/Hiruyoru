# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.5.0] - 2025-12-20

### Added
- **買い物リスト生成機能**: おすすめディナーから必要な食材をAIが自動生成
- **週間レポート機能**: 1週間の食事統計と栄養バランス分析を表示
- 新しい「レポート」タブをナビゲーションに追加

### Changed
- おすすめ画面でディナーを選択すると買い物リストを表示する機能を追加

## [0.4.0] - 2025-12-20

### Added
- ログインUIの改善
- グループ/ファミリー機能
- プッシュ通知機能
- QAテストの追加

### Fixed
- バグ修正とエラーハンドリングの改善

## [0.3.0] - 2025-12-20

### Added
- E2Eテストの追加
- CI/CDパイプラインの設定
- カバレッジレポート機能

### Changed
- テストスイートの改善

## [0.2.0] - 2025-12-20

### Added
- 写真撮影機能
- AI画像解析機能
- 料理認識と自動分類

### Changed
- 基本的な食事記録機能を拡張

## [0.1.0] - 2025-12-20

### Added
- 基本的な食事記録機能（テキスト入力）
- 履歴閲覧機能
- ゲストモード対応
- React Native (Expo) ベースのモバイルアプリ
- Node.js + tRPC バックエンド
- MySQL (TiDB) データベース統合
